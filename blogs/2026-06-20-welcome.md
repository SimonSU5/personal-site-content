# 欢迎来到我的博客

这是第一篇示例文章。

## 关于格式

每篇文章使用 Markdown 格式编写，支持以下 Frontmatter 字段：

- `title`: 文章标题
- `excerpt`: 文章摘要
- `category`: 文章分类
- `tags`: 文章标签（逗号分隔）
- `cover`: 封面图片路径

## 开始写作

在对应的 `YYYY/MM/` 目录下创建新的 Markdown 文件即可。
